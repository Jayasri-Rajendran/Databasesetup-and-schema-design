E-Commerce Database Project

Project Description

This project is an E-Commerce database using Oracle SQL Developer.
It has 4 tables: customers, products, orders, and order_items.
Relationships are created using primary keys and foreign keys.
Sample data is inserted, and queries with JOINs show how tables are connected.
An ER diagram visually represents the tables and their relationships.

Tables Created

Customers – stores customer details
Products – stores product details
Orders – stores order information linked to customers
Order_Items – stores products included in each order

Relationships

One customer → Many orders
One order → Many products
Products can appear in many orders

Queries Demonstrated

Orders with customer names
Order details with product names and quantity
Total amount spent by each customer

Outcome

A well-structured, normalized relational database with proper relationships, data integrity, and query functionality suitable for internship submission.
